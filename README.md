# IssueTracker
Front-End Project

## Using HTML - CSS - JS - JQuery  to build a basic Issue tracker :rocket:

### Pseudo code:
```javascript
Sub Submit-Issue()
  Create an Obj Issue,
  Collect Issue Fields,
  Generate ID,
  Create HTML-Container in IssueList,
  Clear fields
End

Sub Close-Issue()
  Change open lbl to closed
End

Sub Delete-Issue()
  Rm HTML-Container from the IssueList
End
```
